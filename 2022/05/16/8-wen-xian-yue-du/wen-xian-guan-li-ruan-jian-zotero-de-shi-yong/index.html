文献管理软件Zotero的使用

一、Zotero简介

Zotero是一款开源的免费文献管理工具，具有丰富的插件工具。
下载地址为：Zotero | Your personal research assistant

优点主要有：

  • 免费
  • 启动速度快
  • 插件丰富，可以拓展功能
  • 界面简洁（爱了）
  • 强大的英文文献（pdf）分析能力（可以自动获取文献的doi，作者，发表年限等信息，非100%有效）
  • 人性化的标签及笔记功能（也可以添加Markdown插件进行笔记记录）
  • 开源，插件丰富，功能强大，如可设置SCI-HUB网站在软件内下载并加入文献的PDF附件
  • 跨平台同步

缺点主要有：

  • 免费内存只有300M
  • 同步比较复杂，利用坚果云可以实现1G的免费云存储容量

2、设置根目录与数据存储位置

  • 根目录为Zotero文件夹的绝对位置，其他文件均以该路径作为相对路径。

三、添加SCI-HUB下载源

Zotero默认的下载源为Unpaywall，里面能找到的PDF文件并不多，因此需要配置一下下载源。

首选项->高级->设置编辑器，搜索extensions.zotero.findPDFs.resolvers，如下图所示：

删除[]，并将以下代码粘贴进去并点击ok。

{
    "name":"Sci-Hub",
    "method":"GET",
    "url":"https://sci-hub.ren/{doi}",
    "mode":"html",
    "selector":"#pdf",
    "attribute":"src",
    "automatic":true
}

参数解读：

  • “url”:”https://sci-hub.ren/{doi}"：目前可用的域名有.tw、.ren、.si、.shop。（由于Sci-Hub经常更换域名，更换域名之后需要同步修改）
  • Sci-Hub是通过doi下载文献的，因此下载文献必须要有doi。如果doi是空缺的，无法通过PDF解析器免费下载文献。
  • “automatic”:true：设置为true，Zotero会自动下载PDF。

三、导入文献

对于英文文献，可以直接使用DIO的方式导入，或者在浏览器中安装Zotero插件，可以自动抓取。

对于中文文献略显麻烦，但是也还好。首先在知网/万方中下载标准格式的EndNote格式的引用，将其复制到剪切板。
打开Zotero软件，按下 Ctrl+Shift+Alt+I 快捷键即可。
之后，在知网/万方中下载PDF格式的文献，将其以附加文本副本的形式附加到该目录中。

四、安装插件

为Zotero安装一些常用插件，可以提高工作效率。
Zotero 官方维护了一个插件列表，大多数已有的插件都被索引到了。
插件下载地址为：plugins [Zotero Documentation]

插件安装方式：点击工具->插件，将下载好的.xpi插件拖入对话框进行安装。安装后需要重启“Restart now”插件才会生效。

  1. ZotFile
    自动重命名，移动PDF并将其附加到Zotero项目，将PDF从Zotero库同步到您的（移动）PDF阅读器（例如iPad，Android平板电脑等）并从PDF文件中提取注释。

  2. Zotero IF 插件
    显示中科院分区和影响因子（JCR IF 只针对 SCI 期刊论文）。
    Zotero IF v1.1.0
    在本文最后的百度云玩网盘链接中下载xpi插件，将其安装到Zotero中。
    重启软件。
    调出“馆藏目录”和“索取号”，分别对应中科院分区和影响因子（JCR IF 只针对 SCI 期刊论文）。

之后选中要查询的论文（可多选）右击“Update If(s)”即可更新。
